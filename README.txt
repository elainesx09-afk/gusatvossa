ONE ELEVEN — Brand Pack (public/brand)

Conteúdo:
- public/brand/oneeleven-logo.png  (logo completa)
- public/brand/oneeleven-mark.png  (símbolo/mark para favicon)
- public/brand/oneeleven-og.jpg    (opcional para og:image)

Como usar (sem dor):
1) Baixe o .zip
2) Extraia no seu PC
3) GitHub do projeto -> Add file -> Upload files
4) Arraste a pasta "public" extraída para o ROOT do repo e faça commit.

Index.html (favicon + title):
<link rel="icon" type="image/png" href="/brand/oneeleven-mark.png" />
<title>ONE ELEVEN</title>

Open Graph (opcional):
<meta property="og:image" content="/brand/oneeleven-og.jpg" />
